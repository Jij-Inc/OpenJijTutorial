OpenJij チュートリアル¶

OpenJij のチュートリアルへようこそ。

このチュートリアルではOpenJijを使って量子アニーリングをはじめとするアニーリングアルゴリズムの使い方を学んでいきます。 また、現在(2019/5) 1分間/月 だけ使える量子アニーリングデバイスを提供するD-Wave Leap の使い方、OpenJijとのコラボレーションなども学ぶことができます。

本チュートリアルの最終目的はOpenJijを使って、量子アニーリングを使った研究、実証実験などに向けた開発が行えるまでを目指します。

また、量子アニーリングの基礎的なところから最先端の研究についてもコラムなどで紹介していく予定です。
